File Menu:\
1. Press cmd + N to create new File.\
2. Press cmd + S to Save the opened File.\
3. Press cmd + P to Save the opened File.\
4. Press cmd + O to open a file.\
\
Edit Menu:\
1. Press cmd + Z to undo.\
2. Press cmd + Y to redo.\
3. Press cmd + C to copy selected text.\
4. Press cmd + V to paste selected text..\
5. Press cmd + X to cut selected text.\
6. Press cmd + A to select all text.\
7. Press cmd + T to get current date and time.\
7. Press cmd + F to find certain text.\
7. Press cmd + R to replace certain text.\
7. Press cmd + G to take cursor to a specific line number.\
\
Format Menu:\
1. Word wrap makes the text wrap to the size of the window.\
2. Font will change the style of the text Area.\
3. Foreground Color helps set the color of the text.\
\

1. Show/Hide Line, column count.\
